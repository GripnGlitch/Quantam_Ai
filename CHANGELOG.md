# Changelog

## [1.0.0] - 2026-04-15
### Added
- Full chat functionality (global, community, private, groups)
- Polls and pinned messages (5 private, 10 others)
- Leader can mute community chat; admin can mute global chat
- Omnius AI assistant (Bytez, OpenRouter)
- PWA installability with offline fallback
- Role system: Admin, Moderator, Leader, Member
- 4 fixed communities: Vibe Vault, Onyx, R.T.C.S, M.T.T
- Tester badge (admin‑assignable)
- Profile pictures, nickname‑based private messaging
- Translator tool (words ↔ code)
- Developer panel (Eruda) for moderators
- Event calendar with approval queue
- File uploads (images, voice notes, videos)
- Emoji reactions, message deletion, replies
- Online/offline status
- 120‑day auto‑deletion of inactive accounts (admin exempt)
