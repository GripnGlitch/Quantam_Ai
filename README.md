# Resistance

A private, invitation‑only communication platform for closed communities. Built for mobile‑first use.

## Features
- 4 fixed communities: Vibe Vault, Onyx, R.T.C.S, M.T.T
- Global chat (all members) + community chat (per community)
- Private 1‑to‑1 messaging, groups
- Polls, pinned messages, events calendar
- Admin backdoor (audited) for security monitoring
- Omnius AI assistant (Bytez + OpenRouter)
- File attachments, voice notes, videos
- Emoji reactions, message deletion, replies
- PWA installable, offline‑capable

## Deployment
- Frontend: Cloudflare Pages / GitHub Pages
- Backend: Render (Node.js)
- Database: Firestore (asia‑south2)

## Tech Stack
- Frontend: Vanilla JS, Tailwind CSS, Firebase SDK
- Backend: Node.js + Express
- AI: Bytez, OpenRouter
