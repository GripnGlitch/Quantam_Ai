# Security Policy

## Reporting a Vulnerability

If you discover a security issue in Resistance (e.g., unauthorised access to private messages, privilege escalation), **do not open a public issue**. Report directly to @admin.

We will respond within 48 hours.

## Supported Versions
Only the latest `main` branch is actively maintained.

## Notes on Admin Backdoor
The admin can read private messages for security monitoring. This is documented in the [Admin Guide](./docs/ADMIN_GUIDE.md). Users are not notified of this capability.
