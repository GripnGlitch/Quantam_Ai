# Resistance User Guide

## Logging In
- Enter your 9‑digit Access Code and your Member Code (provided by the admin).
- If you don't have an account, use **Sign Up** with a valid Member Code.

## Security
- Messages are stored securely. Administrators may review messages for security monitoring as described in the platform policy.
- Your Access Code is your only credential. Keep it safe.

## Features
- Global Chat, Community Chat, Private Messages, File Attachments, Emoji Reactions.
- Polls, Pinned Messages, Events Calendar, Announcements.
- Omnius AI assistant (Bytez models).

## Troubleshooting
- Forgot your Member Code? Use recovery (if you set security questions).
- Contact admin for account issues.
