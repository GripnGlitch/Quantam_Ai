# Resistance User Guide

## Logging In
1. Open the app.
2. On the **Login** tab, enter your 9‑digit Access Code.
3. Enter your Member Code (Admin: `AMI-021`, Moderator: `OE1-266`, Leader: `LAO-012`, Member: `MEI-021`).
4. If you don’t have an account, go to **Sign Up** and enter your Member Code. You will receive an Access Code. Save it.

## Navigation
- **Dashboard**: Overview.
- **Profile**: Edit your nickname, real name, bio, birthday, hobby, and upload a profile picture.
- **Global Chat**: All Resistance members.
- **My Community Chat**: Only members of your chosen community.
- **Private Messages**: 1‑to‑1 chats (search by nickname) and groups.
- **Announcements**: Read and reply to announcements. Moderators/Admins can post.
- **Events**: View calendar. Members can propose events (need approval).
- **Files**: View all uploaded files.
- **Omnius AI**: Ask AI questions, generate images, transcribe audio.
- **Translator**: Convert text to code symbols and back.
- **Settings**: Customise theme, notifications, etc.

## Sending a Message
- Tap a chat, type your message, press send.
- To attach a file: tap the 📎 icon (image, voice, video).
- To react: tap the emoji button below a message.
- To delete your own message: tap the three dots → Delete (for me or for everyone).
- To reply: tap the reply button on a message.

## Polls
- In global or community chat, click the “Create Poll” button.
- Add a question and up to 5 options.
- Everyone can vote once per poll.
- Results appear after voting.

## Pin Messages
- Click the pin icon on any message to pin it.
- Private chats: up to 5 pinned messages.
- Global / community chats: up to 10 pinned messages.
- Pinned messages stay at the top.

## Groups
- Any user can create a group inside a community.
- Group creator can manage members and delete messages.

## Using Omnius AI
- Select a model from the dropdown.
- Toggle **Random AI** to let the system choose a random model.
- Type your question and wait for the response (streaming).

## Security
- Private messages are end‑to‑end encrypted (only you and the recipient can read them). The admin has a backdoor for security monitoring – this is disclosed in the [Admin Guide](./ADMIN_GUIDE.md).

## Troubleshooting
- **Forgot Member Code?** Use the recovery flow (requires security questions). If you didn’t set them, contact the admin.
- **App not loading?** Clear browser cache or reinstall PWA.
